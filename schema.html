Colonial Collections Data Schema

This document describes the schema used in the PMSAMPO demonstrator. The schema is based on existing Linked Data available through the Colonial Collections endpoint.

Namespace Prefixes

        prefix crm: <http://www.cidoc-crm.org/cidoc-crm/>
        prefix skos: <http://www.w3.org/2004/02/skos/core#>
        prefix wgs84: <http://www.w3.org/2003/01/geo/wgs84_pos#>
        prefix rdfs: <http://www.w3.org/2000/01/rdf-schema#>
        prefix xsd: <http://www.w3.org/2001/XMLSchema#>
        prefix pm: <https://pressingmatter.nl/>

CLASS: crm:E22_Human-Made_Object

Properties

Property Predicate Description
Image pm:shown_by Links the object to an image representation.
Object Type crm:P2_has_type Categorizes the object based on type.
Materials pm:materials_used The materials the object is made of.
Maker pm:maker Identifies the person or entity responsible for producing the object.
Production Place pm:production_place The geographic location where the object was produced.
Production Time Span pm:production_time_span The time period during which the object was created.
Provenance Type crm:P2_has_type Describes how the object was acquired.
Provenance From Actor pm:provenance_from_actor The entity involved in the object's acquisition.
Provenance To Actor pm:provenance_to_actor The entity receiving the object.
Historical Event pm:related_to The event connected to the object's acquisition or transfer.
